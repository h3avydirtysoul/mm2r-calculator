# Калькулятор СМО M/M/2/r с гистерезисными порогами

Численная реализация формул для расчёта стационарных вероятностей,
вероятностных и временны́х характеристик системы массового обслуживания
M/M/2/r с гистерезисной политикой управления числом активных серверов.

Программный комплекс разработан в рамках магистерской ВКР:
"Применение методов теории массового обслуживания для расчёта 
временны́х характеристик электронной очереди" (РУДН, 2026).

**Запуск в браузере:** https://tdarizhapov.github.io/mm2r-calculator/

## Использование

1. Установите параметры системы (λ, μ, L, N, r) с помощью ползунков.
2. Результаты пересчитываются автоматически:
   - стационарные вероятности всех состояний
   - вероятность потери π
   - среднее число заявок n, n̄, Q
   - временны́е характеристики W, T
3. Графики показывают распределение вероятностей и зависимость W(r), T(r).
